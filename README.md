# ECG-Arrhythmia-Detection-MATLAB
ECG Arrhythmia Detection using MATLAB | Signal Processing, R-Peak Detection, Heart Rate Analysis, FFT, MIT-BIH Dataset
Developed an end-to-end ECG signal analysis system in MATLAB using the MIT-BIH Arrhythmia Dataset. The project implements ECG preprocessing, Butterworth filtering, R-peak detection, heart rate estimation, RR interval analysis, FFT-based frequency analysis, and rule-based arrhythmia detection. A live monitoring module simulates real-time cardiac rhythm assessment, providing an interpretable workflow for automated ECG analysis.
